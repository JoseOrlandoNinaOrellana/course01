Write a function that takes two parameters:

1) A constant reference to a vector of integers.

2) An integer (pivot).

and returns an integer indicating how many of the vector's elements are greater than the pivot.

In the main program, data will be read from a text file containing one integer per line. The name of this file will be passed as the first argument when invoking the program. The second argument will be the value of the pivot; if it is not available, the pivot will default to 11.

After calling the function, two lines will be displayed on the standard output:

1. 'Pivot: ' followed by the considered value for the pivot.

2. 'Result: ' followed by the value returned by the function.

In case of any error, the program must return a non-zero error code.

Data files: dataInt1.dat dataInt2.dat dataInt3.dat
